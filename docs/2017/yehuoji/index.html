龙应台《 野火集》摘录

Written by Jiegao with ♥ on July 16, 2017 in 学习笔记

我们的上一代，受战乱和贫穷之苦，期望我们这一代温饱安定。

我们这一代温饱安定了，但是受威权统治之苦，

期望下一代在没有恐惧、没有控制的自由环境中成长。

《野火集》–龙应台
